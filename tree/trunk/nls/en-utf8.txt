# showUsage
2.01:Displays the directory structure of a drive or path.
2.02:\t
2.03:TREE [drive:][path] [/F] [/A]
2.04:\t
2.05:/F   Display the names of the files in each directory.
2.06:/A   Use ASCII instead of extended characters.
2.07:/P   Pause after each page
2.08:/V   Show version information
2.10:/DF  Display file sizes
2.11:/DA  Display file attributes
2.12:/DH  Display hidden and system files (not shown by default)
2.13:/DR  Display results at end
